### 技术栈

前端React Native全家桶, 移动端数据库[Realm](https://github.com/realm/realm-js), 聊天通讯使用环信WebIM-SDK, 状态管理使用Redux,异步状态使用Redux-Thunk,热更新使用[React-Native-Update](https://github.com/reactnativecn/react-native-pushy#readme)。
